## 二、标准咨询流程

### 场景1：新客户咨询（5-10分钟）
```
1. 收到咨询 → 5分钟内响应（模板1）
2. 了解基本信息 → 填写CRM
3. 初步需求分析 → 使用模板3
4. 预约深度咨询 → 确定时间
```

### 场景2：深度咨询（15-20分钟）
```
1. 深入探讨改名原因
2. 分析法律风险
3. 制定初步方案
4. 提供费用估算
```

### 场景3：方案确认（10-15分钟）
```
1. 呈现完整方案（模板5）
2. 解答客户疑问
3. 提供成功案例
4. 促成转化
```

### 场景4：跟进服务（5分钟/次）
```
1. 定期跟进（模板7）
2. 解决问题
3. 提供增值服务
4. 维护客户关系
```

## 四、常见问题快速解决

### 问题1：客户问"改名要多久？"
**快速回答**：
```
改名流程一般需要1-2个月：
1. 材料准备：1-3天
2. 申请提交：1天
3. 政府审批：15-30天
4. 证件更新：7-15天

具体时间取决于您所在城市的办事效率。
```

### 问题2：客户问"改名多少钱？"
**快速回答**：
```
改名费用包括：
1. 政府收费：40-100元（户口本+身份证）
2. 咨询服务：根据服务内容定价
3. 其他证件：视需要更新的证件数量

我们可以根据您的需求提供详细的费用估算。
```

### 问题3：客户担心法律风险
**快速回答**：
```
您的担心很正常！改名确实涉及法律问题。

合法改名需要：
1. 合理的改名理由
2. 完整的证明材料
3. 遵守相关法律法规

我会为您评估法律风险，并提供合规的建议。
```

### 问题4：客户犹豫不决
**快速回答**：
```
我理解您的犹豫，改名是个重要决定。

建议您：
1. 先了解完整的改名流程
2. 看看类似情况的成功案例
3. 考虑改名后的积极变化
4. 我们可以先制定方案，您再决定

不着急，我随时为您解答疑问。
```

## 七、紧急情况处理

### 遇到法律风险客户：
1. 立即使用模板9提醒风险
2. 建议暂停改名流程
3. 提供专业律师咨询建议
4. 记录在CRM备注中

### 遇到情绪激动客户：
1. 保持冷静和专业
2. 使用安抚性语言
3. 提供解决方案
4. 必要时升级处理

### 遇到技术问题：
1. CRM无法访问：检查网络，使用本地备份
2. 模板无法使用：手动编写，保持专业
3. 信息丢失：及时备份重要数据
