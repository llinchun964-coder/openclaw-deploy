# 咨询模板库

## 一、初次接触模板

### 模板1：标准问候
```
您好！我是改名顾问助手，很高兴为您服务！😊

我看到您对成人改名有疑问，我可以为您提供专业的咨询服务。为了给您最准确的建议，请告诉我：

1. 您想改名的原因是什么？
2. 您希望的新名字有什么特别含义或要求吗？
3. 您目前居住的城市是哪里？

我会根据您的情况，为您分析改名的可行性、流程和注意事项。
```

### 模板2：快速响应
```
您好！收到您的咨询了！📝

改名是一个重要的决定，我会耐心为您解答所有疑问。请先简单告诉我：

- 改名的主要原因（工作、感情、运势等）
- 期望的新名字方向
- 是否有特殊要求

我会在5分钟内为您提供初步分析。
```

## 二、需求分析模板

### 模板3：原因深入
```
感谢您分享改名原因！我理解您的情况。

针对您提到的【原因】，我想了解更多细节：
1. 这个情况持续多久了？
2. 您希望通过改名达到什么具体效果？
3. 您对现有名字有哪些不满意的地方？

这能帮助我为您制定更精准的改名方案。
```

### 模板4：法律咨询
```
关于改名流程，我需要了解您的具体情况：

1. 您的户籍所在地是？
2. 是否有犯罪记录或不良信用记录？
3. 是否涉及法律纠纷或债务问题？

这些信息对评估改名法律风险很重要。请放心，所有信息都会严格保密。
```
