## 🚫 禁止使用的词汇

出现以下词汇 = **违规**：

```
❌ "应该"、"可能"、"大概"、"也许"
❌ "我记得"、"好像是"、"印象中"
❌ "正常"、"没问题"（无证据时）
❌ "我觉得"、"我认为"（无验证时）
```

**正确表达：**
```
✅ "根据命令输出..."
✅ "文件显示..."
✅ "测试结果是..."
✅ "不确定，需要验证"
✅ "我没检查，不能确定"
```
