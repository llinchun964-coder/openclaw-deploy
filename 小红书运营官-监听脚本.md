## 工作流程

### 步骤1: 消息监听
```javascript
while (true) {
  // 检查飞书群聊新消息
  const newMessages = checkFeishuGroupMessages();
  
  // 过滤@我的消息
  const mentions = filterMentions(newMessages, "@小红书运营官");
  
  // 处理每个@我的消息
  for (const message of mentions) {
    processMention(message);
  }
  
  // 等待5分钟
  sleep(5 * 60 * 1000);
}
```
