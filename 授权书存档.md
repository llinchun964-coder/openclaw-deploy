## 电子授权流程（飞书表单）

### 方案 A：飞书表单收集
1. 创建飞书表单（授权书内容）
2. 客户在线填写 + 电子签名
3. 自动存档到飞书云文档
4. 导出 PDF 备份

### 方案 B：图片签署
1. 发送授权书图片给客户
2. 客户手写签名后拍照回传
3. 存档到授权书文件夹
4. 命名：`AUTH-日期 - 客户昵称.jpg`

### 方案 C：电子签名平台
1. 使用法大大/上上签等平台
2. 正式电子合同签署
3. 法律效力更强
4. 适合高客单价客户

## 授权书管理规则

1. **命名规范**: `AUTH-YYYYMMDD-序号-客户昵称.pdf`
2. **存放位置**: `/root/.openclaw/workspace/授权书/`
3. **备份策略**: 飞书云文档 + 本地双备份
4. **查阅权限**: 仅 CEO/运营官/业务部可查阅
5. **撤销处理**: 收到撤销通知后，标记为"已撤销"，停止新增使用
