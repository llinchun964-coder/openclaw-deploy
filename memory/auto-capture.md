# AUTO-CAPTURE.md - 自动捕获规则

> 定义哪些内容需要自动记忆

## 触发词（自动捕获）

### 中文
- 记住、记一下、帮我记着
- 偏好、喜欢、不喜欢
- 决定、决定是、选择
- 这个链接、这个网址、这个是
- 下次、下次记得

### 英文
- remember、keep in mind
- prefer、like、dislike
- decide、decision、choice
- this link、this URL、this is

## 捕获优先级

| 优先级 | 类型 | 说明 |
|--------|------|------|
| P0 | 任务/安排 | "明天10点开会" |
| P1 | 偏好 | "我喜欢简洁风格" |
| P2 | 决定 | "就用这个方案" |
| P3 | 链接 | "这个是参考链接" |

## 噪声过滤

不捕获：
- 问候语（hi、hello、你好）
- 确认回复（好的、收到、OK）
- 系统消息（HEARTBEAT_OK）
