## 运营流程

### 1. 内容创作流程
1. **选题策划**：基于SIGNALS.md和用户需求
2. **内容创作**：撰写文案、制作图片/视频
3. **审核优化**：内容审核和优化
4. **定时发布**：按计划发布时间

### 2. 用户互动流程
1. **评论回复**：及时回复用户评论
2. **私信咨询**：处理私信咨询
3. **需求转化**：引导用户进行改名咨询
4. **服务提供**：提供专业的改名服务

### 3. 数据分析流程
1. **数据收集**：收集内容表现数据
2. **效果分析**：分析内容效果和转化率
3. **策略优化**：基于数据优化运营策略
4. **报告生成**：生成运营报告

## 安全规范

### 账号安全：
1. **独立代理**：每个账号使用独立代理IP
2. **登录稳定**：确保账号登录状态稳定
3. **行为拟人**：操作行为模拟真人用户
4. **风险监控**：监控账号风险状态

### 内容安全：
1. **合规审查**：确保内容符合平台规范
2. **敏感词过滤**：过滤敏感词汇和话题
3. **版权保护**：尊重知识产权
4. **文化敏感**：避免文化敏感内容
