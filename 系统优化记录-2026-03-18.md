```json
  "maxHistoryShare": 0.3,

  "memoryFlush": {
    "enabled": true,
    "forceFlushTranscriptBytes": "100kb"
  }
}
```

---

## 配置说明

- **mode: "default"** - 更激进的压缩模式（之前是 "safeguard"）
- **maxHistoryShare: 0.3** - 历史记录最多占上下文 30%
- **recentTurnsPreserve: 10** - 保留最近 10 轮对话
- **forceFlushTranscriptBytes: "100kb"** - 超过 100KB 自动刷新内存到 memory 文件
